# D3 Functions

- `.sort(array)`
- `.min(array)`
- `.max(array)`
- `.sum(array)`
- `.mean(array)`
- `.median(array)`
- `.extent(array)` returns `[d3.min(array), d3.max(array)]`
- `.variance(array)`
- `.deviation(array)`
- `.quantile(array,p)` (`p` should be a single value)
